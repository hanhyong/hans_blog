망각곡선 퀴즈 에디터 안내

이 도구는 선생님이 직접 만든 문제를 망각곡선에 맞춰 반복 학습할 수 있게 돕는 브라우저 기반 퀴즈 에디터입니다. 왼쪽에서 문제를 작성하면 오른쪽에 카드가 자동으로 만들어지고, 풀이 모드에서는 시험/카드 연습으로 학습할 수 있습니다.

  • ① 편집 모드 – #로 문항 제목, - / > / * / !로 답·보기 작성, 실시간 카드 프리뷰
  • ② 풀이 모드 – 시험(한 번에 채점) / 카드 연습(한 문제씩 확인) 전환
  • ③ 문제 유형 – 단답형, 순서형, 선지형, OX형까지 지원
  • ④ 우선도/망각곡선 – ⭐ 중요 체크 + 마지막 풀이 시점으로 우선도 자동 계산
  • ⑤ 범위/섞기 – 전체 / 상위 50% / 상위 20% + 무작위 섞어서 세트 구성
  • ⑥ 기록 초기화 – 문제 내용은 유지한 채 풀이 기록만 초기화 또는 전체 초기화
  • ⑦ 링크 공유 – 현재 문제와 통계 상태를 URL 해시에 저장해서 그대로 복원
  • ⑧ GPT 연동 – 아래 프롬프트를 사용하면 교과 내용을 넣고 자동으로 문항 세트를 생성

문법 요약

각 문항은 #제목 아래에 답/보기 줄을 적고, 문항 사이에는 빈 줄 한 줄을 둡니다. 

① 단답형
#세포호흡의 최종 전자 수용체는?
  - 산소
  - O₂ / O2

② 순서형
#광계 II와 I의 작동 순서를 쓰시오
  >광계 II
  >광계 I

③ 선지형 (객관식, 여러 개 정답 가능)
#식물세포에서만 볼 수 있는 구조는?
  *엽록체          <-- 정답: * 한 개 이상 가능
  !중심소체        <-- 오답: ! 또는 -
  !리소좀
  -미토콘드리아

④ OX형
#물분자는 [1/2/3]개의 산소 원자와 [2/1/4]개의 수소 원자로 이루어져 있[다/지지 않는다].:: OX

- 제목 끝에 ":: OX" 를 붙입니다.
- 대괄호 [ ] 안의 값들 중 "첫 번째"가 참 문장일 때의 표현입니다.
- 나머지 값들은 거짓 문장을 만들 때 랜덤으로 사용됩니다.
- 프로그램이 매번 일부 괄호를 오답 값으로 바꾸어 O/X 문제를 출제합니다.
